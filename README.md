# Borneo Endemic Ant 🐜

**Live exotic ants, insects & reptiles from the rainforests of Borneo. Shipped worldwide.**

🌐 [Live Website](https://astrorawak.github.io/borneo-endemic-ant/)

## Stack
- React 18 + TypeScript + Vite
- Tailwind CSS
- Zustand (localStorage persistence)
- React Router DOM v6
- GitHub Pages (auto-deploy via Actions)

## Local Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
